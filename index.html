Formatting Strings Like A Boss

Or at least like an assistant manager!

So how do you format a string?

Formatting a string takes a little finesse. Unlike HTML, where you use break points or a pre-formatted tag, Javascript actually has what is called an escape sequence built in!

It includes lots of neat stuff, like \b for a backspace, \t to tab a paragraph in, \n to create a new line.
